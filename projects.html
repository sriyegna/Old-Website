  • 07 2015

    Pikaptcha Language: Python

    When Pokemon Go was released, the hype was too large to handle. When the in-game tracker broke, many players resorted to using "maps". These maps were achieved by falsifying the Pokemon Go API and utilizing spare Pokemon Go Trainer accounts to "walk" through the streets to see if they notice any pokemon. As you can imagine, this requires a large amount of accounts to scan a large region so many Pokemon Go Account creators were being made. Eventually when they added recaptcha into the registration page, the other scripts could not bypass that. Only one did, and he charged everyone $10 for the script. So I created a python script that emulates the browser (headless or using chrome) to go through the registration, and if they opted to use 2captcha, then Pikaptcha would automatically solve their captchas for them.Check it out!
